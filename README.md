Research project exploring agentic AI design patterns on the MMLU benchmark. I implemented and evaluated multiple LLM-based approaches—including a baseline workflow, ReAct, tool-calling, and a hybrid ReAct + tool-calling system—to analyze how different agent architectures affect performance across domains such as computer science, philosophy, and general knowledge.

The results showed that more complex agent designs do not always lead to better performance. In some cases, added reasoning steps or tool use introduced unnecessary overhead or overthinking that reduced accuracy. This project reinforced the importance of rigorous evaluation, iterative testing, and understanding tradeoffs when designing AI-powered systems rather than assuming increased complexity improves outcomes.
